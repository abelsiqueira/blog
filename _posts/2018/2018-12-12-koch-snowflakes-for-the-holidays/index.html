Koch snowflakes for the holidays

Code for these images.

I hope you're familiar with the Koch curve fractal or snowflake. It essentially consists taking a line segment, splitting it in three, and substituting the middle part by two segments that form an equilateral triangle without the base. From one segment you obtain four. For each new segment, repeat the process.

Images:

The most important aspect of the koch line is that it looks awesome. Furthermore, you can do it for any image that is a collection of segments. In particular, regular polygons, both outward and inward.

Another way to define the four new segments is this: Given the endpoints of the segment P1P_1P1​ and P2P_2P2​, we define v⃗=P1P2⃗\vec{v} = \vec{P_1P_2}v=P1​P2​​. The five points that define the four new segments, in order, are P1P_1P1​, P1+13v⃗P_1 + \frac{1}{3}\vec{v}P1​+31​v, P1+12v⃗+αRv⃗P_1 + \frac{1}{2}\vec{v} + \alpha R\vec{v}P1​+21​v+αRv, P1+23v⃗P1 + \frac{2}{3}\vec{v}P1+32​v and P2P_2P2​, where α=3/6\alpha = \sqrt{3}/6α=3​/6. A simple thing one can do is change the value of α\alphaα to obtain different images:

Using a diamond with α=1.25\alpha = 1.25α=1.25.

Using a square with α=1.2\alpha = 1.2α=1.2.

Using polygons with NNN equals to 3, 4 and 5 sides, and α=3N/5\alpha = 3\sqrt{N}/5α=3N​/5.

Using a line segment from the center to the vertex and back and to next vertex, etc., with α=1\alpha = 1α=1.
